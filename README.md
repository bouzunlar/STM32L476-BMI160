# Real-Time Activity Detection System Using BMI160 IMU on STM32 with FreeRTOS

## Description
This project implements a real-time circular motion detection system on the STM32L476 Nucleo-64 board using the BMI160 IMU sensor and FreeRTOS.
## Features
- I2C communication with BMI160
- FreeRTOS-based architecture
- Circular motion detection

## Hardware
- STM32L476
- BMI160 IMU
